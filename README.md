
# Contour Plot AROPE España

Este proyecto muestra un Contour Plot del riesgo de pobreza o exclusión social (AROPE) en España.

## Técnica
Se utiliza un Contour Plot para representar la relación entre tres variables:
- Año (eje X)
- Edad media (eje Y)
- Tasa AROPE (%) (color)

## Datos
Los datos provienen del Instituto Nacional de Estadística (INE).

Se han transformado:
- Reorganización a formato largo
- Conversión de grupos de edad a valores numéricos (punto medio)

## Resultados

- Mayor riesgo en población joven
- Disminución del riesgo con la edad
- Diferencias entre hombres y mujeres
